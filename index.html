StepUp

StepUp is a command-line tool that helps other projects to manage its semantic versioning increment and its release notes.

Check the RubyGems Page View on GitHub

Introduction

StepUp is a tool to manage semantic versioning. That means you can bump the version of your project, for example, from v3.0.11 to v4.0.0, check the current version, summarize all the history of versions in a changelog and more.

StepUp is based on source control management features (i.e. tags, branches, commits, notes etc), so it doesn’t need to keep files with the current version (but it supports it, if you want), it has visibility of all history of changes and versions (doesn’t matter if they are spread in many different branches), which adds a variety of possibilities in terms of management, depending on your project needs.

Instalation and Usage

First of all, your installed git-scm version must be v1.7.1 or higher. This minimal version is required because of git notes command-line support.

cd your_repo_root/repo_name
git --version
git version 1.7.9.5
git notes list; echo $?
0

Basic usage:

gem install step-up
Successfully installed step-up-0.9.5
1 gem installed
cd your_repo_root/repo_name
stepup
v0.2.0+3

Consider that your project has a tag named "v0.2.0". The "+3" part of the output above means the project has three commits since last version tag. The format of the version is totally customizable, and we will cover in more detail later.

Authors and Contributors

In 2010, Marcelo Manzan (@kawamanza) had the idea along with Eric Fer (@ericfer) to improve the way they generate documentation on projects. They were based on the reference documentation that was being created on the site SemVer.org, and designed a way to automate incrementing versions of projects based on scope of changes made. So the StepUp Project emerged. A consolidated utility that helps programmers, deployers and other stakeholders in the project’s life cycle.

Special thanks to Lucas Fais (@lucasfais) for his participation in drafting some ideas and the implementation of some features.
